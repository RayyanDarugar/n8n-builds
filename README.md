This past summer, I've really enjoyed building workflows in n8n. And by enjoyed, I mean agonizingly debugged for hours at a time.

Either way, I use them because I've found that they're a great way to handle specific tasks with granularity and control. Everything is agentic nowadays but leaving it all to claude is expensive, silly, and honestly not the most accessible. Sometimes all you need is a (still expensive) api key, and the creativity in your heart.

I probably spent more time making automations here than it wouldve taken me to manually do the tasks. The good thing is, I already built some for you! Feel free to use, clone, and reference anything here. Detailed README guides for how to customize/plug and play each piece will be in each folder.

These will need access to n8n, plus accounts with the other services that each workflow uses. If you use a different tool like Make.com or Zapier, theoretically you can import the JSONs the same way but there will be a lot of bugs to work out.
